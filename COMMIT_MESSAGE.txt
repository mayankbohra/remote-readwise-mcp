feat: Add FastMCP remote server for Claude.ai web app support

Implements a production-ready Python FastMCP server that enables Readwise
integration for Claude.ai web application users. This allows access to all
Readwise features from any device without local installation.

## Major Changes

### New Python FastMCP Server (`server/`)
- Complete rewrite in Python using FastMCP framework
- All 13 tools ported (6 Reader + 7 Highlights)
- HTTP transport for remote access
- Bearer token API authentication
- Health check endpoint
- CORS configuration for Claude.ai

### Security Features
- API key authentication middleware
- Environment-based secret management
- Two-layer security (MCP key + Readwise token)
- HTTPS enforcement via deployment platform

### Deployment Infrastructure
- Docker containerization (multi-platform support)
- Render deployment configuration (`render.yaml`)
- Support for Railway, Fly.io, Cloud Run
- Auto-scaling and health monitoring

## Usage for Claude.ai

1. Deploy server to Render/Railway/Fly.io
2. Configure environment variables (READWISE_TOKEN, MCP_API_KEY)
3. Add custom connector in Claude.ai Settings → Connectors
4. Access Readwise from any device via Claude.ai
